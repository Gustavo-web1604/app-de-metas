# Gerenciador de Tarefas e Metas - Versão Desktop

Um aplicativo desktop para ajudar no gerenciamento e acompanhamento de metas diárias, desenvolvido com Electron.

Esse projeto começou como uma versão de linha de comando (CLI) em Node.js, construída a partir dos ensinamentos do evento/curso da Rocketseat. Com o tempo, resolvi evoluir essa base e transformei o app em um programa desktop de verdade, com janela própria e interface gráfica.

## Funcionalidades

A aplicação abre em uma janela própria e é organizada em abas, onde é possível:

- Cadastrar novas metas.
- Listar todas as metas, separadas por abas: Todas, Realizadas e Abertas.
- Marcar e desmarcar metas como concluídas.
- Deletar metas que não fazem mais sentido.
- Persistência de dados: tudo é salvo automaticamente, então o progresso nunca se perde.

## Tecnologias Utilizadas

- JavaScript / Node.js
- Electron: para transformar o app em um programa desktop nativo.
- HTML / CSS: estrutura e estilo da interface.
- electron-builder: usado para gerar o instalador executável.

## Pensando no Futuro (Próximos Passos)

Essa versão desktop é só mais uma etapa da evolução do projeto, que começou no terminal. A lógica e a estrutura construídas desde o início continuam servindo de base para os próximos passos, que incluem:

1. Transformar em uma API REST.
2. Criar uma versão web, acessível direto pelo navegador.
3. Pensar em sincronização entre dispositivos.
